Las 30 Primeras Derivadas Inmediatas

  1. $$\frac{d}{dx} (c) = 0$$
  2. $$\frac{d}{dx} (x^n) = nx^{n-1}$$
  3. $$\frac{d}{dx} (e^x) = e^x$$
  4. $$\frac{d}{dx} (\ln |x|) = \frac{1}{x}$$
  5. $$\frac{d}{dx} (\sin x) = \cos x$$
  6. $$\frac{d}{dx} (\cos x) = -\sin x$$
  7. $$\frac{d}{dx} (\tan x) = \sec^2 x$$
  8. $$\frac{d}{dx} (\arcsin x) = \frac{1}{\sqrt{1-x^2}}$$
  9. $$\frac{d}{dx} (\arccos x) = -\frac{1}{\sqrt{1-x^2}}$$
  10. $$\frac{d}{dx} (\arctan x) = \frac{1}{1+x^2}$$
  11. $$\frac{d}{dx} (\sinh x) = \cosh x$$
  12. $$\frac{d}{dx} (\cosh x) = \sinh x$$
  13. $$\frac{d}{dx} (\tanh x) = \text{sech}^2 x$$
  14. $$\frac{d}{dx} (\text{csch} x) = -\text{csch} x \coth x$$
  15. $$\frac{d}{dx} (\text{sech} x) = -\text{sech} x \tanh x$$
  16. $$\frac{d}{dx} (\coth x) = -\text{csch}^2 x$$
  17. $$\frac{d}{dx} (a^x) = \ln(a) \cdot a^x$$
  18. $$\frac{d}{dx} (\log_a x) = \frac{1}{x \ln(a)}$$
  19. $$\frac{d}{dx} (\sin^{-1} x) = \frac{1}{\sqrt{1-x^2}}$$
  20. $$\frac{d}{dx} (\cos^{-1} x) = -\frac{1}{\sqrt{1-x^2}}$$
  21. $$\frac{d}{dx} (\tan^{-1} x) = \frac{1}{1+x^2}$$
  22. $$\frac{d}{dx} (\sinh^{-1} x) = \frac{1}{\sqrt{1+x^2}}$$
  23. $$\frac{d}{dx} (\cosh^{-1} x) = \frac{1}{\sqrt{x^2-1}}$$
  24. $$\frac{d}{dx} (\tanh^{-1} x) = \frac{1}{1-x^2}$$
  25. $$\frac{d}{dx} (\sec x) = \sec x \tan x$$
  26. $$\frac{d}{dx} (\csc x) = -\csc x \cot x$$
  27. $$\frac{d}{dx} (\text{arcsec} x) = \frac{1}{|x| \sqrt{x^2-1}}$$
  28. $$\frac{d}{dx} (\text{arccsc} x) = -\frac{1}{|x| \sqrt{x^2-1}}$$
  29. $$\frac{d}{dx} (\text{cot} x) = -\csc^2 x$$

Ecuaciones de la Parábola

Forma Canónica

  • $$y = ax^2$$
  • $$x = ay^2$$

Forma Principal

  • $$y^2 = 4ax$$
  • $$x^2 = 4ay$$

Variante con el Vértice en el Punto (h, k)

  • $$y - k = a(x - h)^2$$
  • $$x - h = a(y - k)^2$$

Ecuación General

  • $$Ax^2 + Bxy + Cy^2 + Dx + Ey + F = 0$$

Otras Variantes

  • $$y = ax^2 + bx + c$$
  • $$x = ay^2 + by + c$$
  • $$y^2 = 4ax + 2bx + c$$
  • $$x^2 = 4ay + 2by + c$$

Circunferencia con centro en (h,k):

\begin{equation} (x - h)^2 + (y - k)^2 = r^2 \end{equation}

Hipérbola con centro en (h,k):

\begin{equation} \frac{{(x - h)^2}}{{a^2}} - \frac{{(y - k)^2}}{{b^2}} = 1 \end{equation}
