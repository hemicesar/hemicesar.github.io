Ecuaciones Importantes en Álgebra Lineal

  1. $$\mathbf{Ax} = \mathbf{b}$$
  2. $$\mathbf{A}(\mathbf{Bx}) = (\mathbf{AB})\mathbf{x}$$
  3. $$\mathbf{A}(\mathbf{B} + \mathbf{C}) = \mathbf{AB} + \mathbf{AC}$$
  4. $$\mathbf{A}(\mathbf{B}^T) = (\mathbf{A}^T)(\mathbf{B}^T)$$
  5. $$\mathbf{A}^{-1}\mathbf{A} = \mathbf{I}$$
  6. $$\text{det}(\mathbf{A}) = \text{det}(\mathbf{A}^T)$$
  7. $$\mathbf{A}^T\mathbf{A} = \mathbf{A}\mathbf{A}^T$$
  8. $$\text{Rank}(\mathbf{A}) + \text{Rank}(\mathbf{B}) - \text{Rank}(\mathbf{AB}) \leq n$$
  9. $$\mathbf{A}^T\mathbf{A}^{-1} = (\mathbf{A}^{-1})^T\mathbf{A}^T = (\mathbf{A}^T\mathbf{A})^{-1}$$
  10. $$\text{Rank}(\mathbf{A}) = \text{Rank}(\mathbf{A}^T\mathbf{A})$$
  11. $$\mathbf{Ax} = \mathbf{0} \text{ tiene solución única si y solo si } \text{Rank}(\mathbf{A}) = n$$
  12. $$\mathbf{A}^{-1} = \frac{1}{\text{det}(\mathbf{A})} \text{adj}(\mathbf{A})$$
  13. $$\text{det}(\mathbf{A}^T) = \text{det}(\mathbf{A})$$
  14. $$\text{Rank}(\mathbf{A} + \mathbf{B}) \leq \text{Rank}(\mathbf{A}) + \text{Rank}(\mathbf{B})$$
  15. $$\text{Rank}(\mathbf{A}^T\mathbf{A}) = \text{Rank}(\mathbf{A})$$
  16. $$\text{Rank}(\mathbf{AB}) \geq \text{Rank}(\mathbf{A}) + \text{Rank}(\mathbf{B}) - n$$
  17. $$\text{Nullity}(\mathbf{A}) + \text{Rank}(\mathbf{A}) = n$$
  18. $$\text{det}(\mathbf{A}^{-1}) = \frac{1}{\text{det}(\mathbf{A})}$$
  19. $$\mathbf{A}(\mathbf{A}^T\mathbf{A})^{-1}\mathbf{A}^T = \mathbf{P}_\mathbf{A}$$
  20. $$\mathbf{A}^{-1} = \frac{1}{\text{det}(\mathbf{A})}\text{adj}(\mathbf{A})$$

Ecuaciones Diferenciales Importantes

  1. $$\frac{dy}{dx} = f(x)$$
  2. $$\frac{d^2y}{dx^2} = f(x)$$
  3. $$\frac{d^2y}{dx^2} + p(x)\frac{dy}{dx} + q(x)y = 0$$
  4. $$\frac{dy}{dt} = k(y - L)$$
  5. $$\frac{d^2y}{dt^2} = -g$$
  6. $$\frac{d^2y}{dt^2} = -\frac{k}{m}y$$
  7. $$\frac{d^2y}{dx^2} = -\omega^2y$$
  8. $$\frac{d^2y}{dx^2} + \omega^2y = 0$$
  9. $$\frac{d^2y}{dt^2} + \frac{c}{m}\frac{dy}{dt} + \frac{k}{m}y = 0$$
  10. $$\frac{d^2y}{dx^2} - p^2y = 0$$
  11. $$\frac{d^2y}{dx^2} + a^2y = 0$$
  12. $$\frac{d^2y}{dx^2} - \lambda^2y = 0$$
  13. $$\frac{d^2y}{dx^2} + \frac{2}{x}\frac{dy}{dx} - \frac{n(n+1)}{x^2}y = 0$$
  14. $$\frac{d^2y}{dx^2} - \frac{1}{x}\frac{dy}{dx} + \frac{1}{x^2}y = 0$$
  15. $$\frac{d^2y}{dt^2} = c^2\frac{d^2y}{dx^2}$$
  16. $$\frac{d^2y}{dt^2} - c^2\frac{d^2y}{dx^2} = 0$$
  17. $$\frac{d^2y}{dx^2} + \lambda^2y = \delta(x-a)$$
  18. $$\frac{d^2y}{dx^2} - p(x)\frac{dy}{dx} + q(x)y = f(x)$$
  19. $$\frac{d^2y}{dt^2} = -k^2y$$
  20. $$\frac{d^2y}{dx^2} + k^2y = 0$$
